# 💻 Enterprise Code RAG Assistant

An AI-powered code assistant that can:
- 📦 Index any GitHub repository
- 🔎 Answer questions about the codebase
- 🐞 Detect bugs and suggest fixes

---

## 🚀 Features

- 🔍 Retrieval-Augmented Generation (RAG)
- 🧠 Semantic search over code
- 🐞 Code bug detection & correction
- ⚡ Fast indexing (optimized)
- 💬 Interactive Streamlit UI

---

## 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- ChromaDB
- HuggingFace Embeddings
- Groq LLM (LLaMA 3.1)
